KDL-JS

A JavaScript library for the KDL Document Language.

Install

npm install kdljs

Usage

Right now, only parsing is supported:

const parseKdl = require('kdljs')
parseKdl(`// Nodes can be separated into multiple lines
title \
  "Some title"
